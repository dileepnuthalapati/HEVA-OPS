🚀 BUILD ON YOUR MACHINE (5 MINUTES):

  1. Install Android Studio + Java JDK 17
  2. Open terminal in: HEVA-OPS/frontend/
  3. Run: npx cap open android
  4. In Android Studio: Build → Build APK(s)
  5. Find APK: android/app/build/outputs/apk/debug/
